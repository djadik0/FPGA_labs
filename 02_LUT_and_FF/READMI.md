# Лабораторная работа 2. Таблицы истинности LUT и FF

## Цель работы

Изучить низкоуровневое проектирование логики для FPGA Xilinx 7 с использованием библиотечных примитивов Vivado.  
В рамках лабораторной работы необходимо:
- реализовать комбинационную логику с помощью примитивов LUT;
- реализовать запоминание результата с помощью примитива D-триггера;
- проверить работу схемы в моделировании с помощью testbench.

## Краткая теория

Vivado позволяет использовать встроенные библиотечные элементы FPGA напрямую через примитивы.  
В данной лабораторной работе рассматриваются следующие типы примитивов:

- **LUT1-LUT6** — примитивы таблиц истинности для реализации комбинационной логики;
- **CARRY4** — примитив цепочки переноса для арифметических операций;
- **FDCE / FDPE / FDRE / FDSE** — примитивы D-триггеров с различными типами сброса и управления.

Использование примитивов позволяет лучше понять, как логика проектируемой схемы отображается на реальные аппаратные ресурсы FPGA.

## Что было сделано в работе

В ходе лабораторной работы была реализована комбинационная функция с использованием примитивов LUT.  
После этого результат работы комбинационной схемы был сохранён с помощью примитива D-триггера.

Также был написан простой testbench на `initial` блоках для проверки корректности работы схемы в моделировании.

## Выполненные индивидуальные задания

### Часть 1 — вариант 6

Была реализована функция:

```text
y[3:0] = a[3:0] ^ b[3:0] ^ (~c[3:0])
```

Реализация выполнена с использованием примитивов LUT.

### Часть 2 — вариант 2

Было реализовано запоминание результата первой части с помощью примитива **FDRE**.

Примитив `FDRE` представляет собой D-триггер с тактовым входом, разрешением записи и синхронным сбросом.  
В данной работе он использовался для регистрации результата комбинационной логики, полученной в первой части задания.
