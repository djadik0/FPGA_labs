
# Лабораторная работа 1. Язык Tcl в Vivado

## Цель работы

Изучить основы языка Tcl и его применение для автоматизации работы в САПР Vivado:
- создание проекта из исходных файлов,
- запуск стандартных этапов проектирования,
- открытие графических окон анализа результатов,
- автоматизация генерации bitstream и прошивки отладочной платы.

## Краткая теория

**Tcl (Tool Command Language)** — скриптовый язык высокого уровня, который широко используется для автоматизации работы в FPGA/ASIC САПР.

В среде **Vivado** Tcl позволяет:
- автоматически создавать проекты,
- добавлять исходные файлы, testbench и constraints,
- запускать synthesis, implementation и bitstream generation,
- открывать нужные окна анализа в графическом интерфейсе,
- работать с проектом без использования графического интерфейса.

## Что было сделано в работе

В рамках лабораторной работы был создан проект в Vivado, после чего сгенерирован Tcl-скрипт для автоматизированного восстановления проекта.

В ходе выполнения работы был изучен сгенерированный Vivado Tcl-скрипт:
- найдены секции создания проекта,
- найдены команды добавления RTL, testbench и XDC-файлов,
- найдено место выбора **part number** FPGA,
- изучены команды запуска основных этапов проектирования.

Также была изучена работа **Tcl Console** в Vivado и принцип, по которому действия в графическом интерфейсе соответствуют Tcl-командам.
